# wechat-auto-send
微信自动回复(接入图灵机器人)  每日发送天气预报

## itchat
itchat是一个开源的微信个人号接口，本程序通过其完成数据的传送

## auto-wechat-response
通过itchat和图灵机器人接收下载微信的信息、图片、音频、视频，并且接入图灵机器人回复消息。

## auto-send-weather
通过itchat和图灵机器人每日定时发送天气预报到特定个人和群聊。
